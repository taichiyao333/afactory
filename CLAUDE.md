# CLAUDE.md — afactory リポジトリ作業規約

このファイルはリポジトリ全体の作業規約です。**どのタスクに入る前にも、必ず最初にこれを読み、次に以下3ファイルを熟読してください。**

## 🔴 作業開始前に必ず読むファイル

対象テンプレート(例: `templates/ace-factory/`)の中の:

1. **HANDOFF.md** — 残TODO / 既知の制約 / 素材の出所 / 未解決事項
2. **README.md** — ブランド定義(声・トーン・色・タイポ・ロゴ・イメージ方針)
3. **SKILL.md** — このテンプレを使ってサイトを組む手順

この3ファイルを読まずにコード編集・ファイル作成に入ることは禁止です。
読み終えたら **作業方針(どのファイルを触るか / どんな順序で進めるか / 不明点は何か)を先に返答してから** 実装に着手してください。

## 🔴 ブランド禁則 (Ace Factory)

これはこのリポジトリで扱う唯一のブランド "Ace Factory" の絶対ルールです。`templates/ace-factory/` 配下および `apps/` 配下の実装は、以下を破ってはいけません。

- **色**: 背景は黒 (`#0B0B0B`) + 白 (`#FFFFFF`) の2色が基調。アクセントは **オレンジ `#E95A0C` のみ**。他のアクセントカラー(青・緑・紫など)の追加は禁止。グラデーションも使わない。
- **アイコン**: **最小限**。UIの意味が文字で伝わる箇所には置かない。使う場合もラインアイコン1種類に揃え、装飾目的の絵文字・イラスト追加は不可。
- **角丸**: **2px 以下**(`--radius-sm: 2px` が最大)。カード・ボタンは基本直角。丸ボタン・ピル形状は使わない。
- **シャドウ**: **禁止**。`box-shadow` を新規に足さない。階層は線 (`1px solid`) と余白で表現する。
- **タイポ**: 欧文は Cormorant Garamond(Display) / Inter(UI)、和文は Noto Serif JP(見出し) / Noto Sans JP(本文)。他フォントへの差し替えは禁止。
- **余白**: 4の倍数スケール(`--space-1` = 4px から `--space-16` = 128px)。独自数値を直書きしない。
- **写真**: ハイコントラスト・低彩度のドキュメンタリー調。過度な補正・ボケ加工・フィルター合成は使わない。

詳細値は `templates/ace-factory/colors_and_type.css` のトークンが正。CSS変数を無視したハードコード値(`color: #333` など)は禁止。

## 🔴 作業対象の識別

ディレクトリごとに意味と扱いが違います。混同しないこと。

| ディレクトリ | 役割 | 編集ポリシー |
|---|---|---|
| `templates/<name>/` | **テンプレ定義**(デザインシステム + UIキット + コンテンツ雛形) | ブランドルールそのもの。編集は慎重に。トークン・コンポーネントの変更は影響範囲を明示してから。 |
| `apps/<name>/` | **本番実装**(実際にデプロイされるサイト) | テンプレを import/参照して組み上げる。ブランドルールの再定義はしない(必ず templates 側に揃える)。 |
| ルート直下 | リポジトリ全体の規約・説明 | `CLAUDE.md` / `README.md` 以外を足すときは理由を明記。 |

新しい店舗ページを増やすだけなら `templates/ace-factory/content/sites/<slug>.json` を追加するだけで済むよう設計されています。安易に新しいコンポーネントや新しい色を導入しないでください。

## 🔴 作業方針の共有

実装に入る前に、以下を毎回必ずユーザーに返してから着手してください:

1. どのテンプレート / アプリが対象か
2. どのファイルを触るか(新規 / 編集 / 削除)
3. どの順序で進めるか
4. ブランド禁則に抵触しそうな判断があれば、その相談
5. 不明点・前提置きの確認

承認を得てからコード編集に入ること。勝手に大量のファイルを作らない。

## メモ

- GitHub 経由で Claude Code Web が操作する想定。初回プッシュはこのリポジトリが空の状態から一括で入ります。
- 新しいブランド/テンプレを足す場合は `templates/<new-name>/` に並列配置。既存の `ace-factory/` 側に混ぜない。

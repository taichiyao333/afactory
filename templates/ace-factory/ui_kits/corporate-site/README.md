# UI Kit — Ace Factory Corporate / Venue Site

A high-fidelity recreation of the Top page comp (`uploads/A_Factory_HP_Top_260417.pdf`). Components are split by section so they can be reused or reordered.

## Files
- `index.html` — entry, loads fonts + CSS + all JSX, renders the whole Top page
- `Nav.jsx` — fixed top nav with type-only links + logo
- `Hero.jsx` — full-bleed interior photograph with centred EN/JP display pair
- `SectionHeading.jsx` — the two-line EN+JP heading motif
- `PhilosophySection.jsx` — "Our Philosophy" body block
- `ServiceSection.jsx` — "Our Service" block + ALL VENUES list
- `MessageSection.jsx` — CEO Message with portrait
- `InterviewSection.jsx` — cast card grid with vertical-tategaki motto
- `RecruitBand.jsx` — the full-bleed orange Recruit band
- `NewsSection.jsx` — news list
- `AccessSection.jsx` — 6-venue access block
- `Footer.jsx` — footer with venue wordmarks + copyright

## Usage

Open `index.html` directly. Everything is rendered via Babel-standalone so no build step is needed.
